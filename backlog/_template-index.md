# Backlog

프로젝트 전체 백로그 단일 소스 문서.

상태 기준:

- `Now`: 다음 작업으로 바로 착수
- `Next`: 현재 스코프 다음 우선순위
- `Later`: 아이디어/중장기 과제

ID 규칙:

- 형식: `B-001`, `B-002` ...
- 상태 이동 시 ID는 유지한다.
- 완료 시 항목을 삭제하지 않고 `[x]`로 표시해 이력을 남긴다.

## Now

- [ ] `B-001` <지금 바로 착수할 작업>  
  ref: `specs/<feature-spec>.md`

## Next

- [ ] `B-002` <다음 우선순위 작업>  
  ref: `specs/<feature-spec>.md`

## Later

- [ ] `B-003` <중장기/아이디어 작업>  
  ref: `specs/<feature-spec>.md`
